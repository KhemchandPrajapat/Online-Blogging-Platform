Blogging Platform

A dynamic and user-friendly web application that allows users to create, edit, delete, and view blogs.
The platform includes a rich text editor, like and comment features, and complete profile management for each user.
Built using the MERN stack, this application provides a smooth blogging experience with modern UI and responsive design.

How It Works

1. User registers or logs in to the platform.


2. Create a new blog using the rich text editor.


3. Blogs are displayed in a feed with titles, images, and previews.


4. Users can like, comment, and interact with posts.


5. Each user has a profile dashboard showing their blogs and activity history.
